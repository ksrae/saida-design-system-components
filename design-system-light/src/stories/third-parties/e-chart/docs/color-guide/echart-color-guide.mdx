# EChart Color Guide
The color token below will be automatically filled according to the Echart rule.<br/>
(Status color should be applied directly if necessary.)<br/><br/>

Echart does not have a separate token scss file.<br/>
Please use the code below with the synergy theme.<br/>
When you install sy-components, it is already applied.<br/><br/>

If you want to change the colors.<br/>
You can change the theme through the ecart-theme.js file.<br/>

## Overview
Data visualization is the representation of information in pictorial or graphical format, such as charts, graphs, maps, and diagrams. <br/>
These visuals aid in the communication of complex data so that insights can be more easily drawn.
Because color can affect our perception of information, the appropriate use of color is critical in making a data visualization successful.<br/>
