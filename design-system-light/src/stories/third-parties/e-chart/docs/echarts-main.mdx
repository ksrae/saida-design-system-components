# E-Chart Style Wrapper
The style wrapper for Echart.<br/>
If you would like to apply the theme that fits the DC component to the Echart, please add class as below.
We use the JS version to apply a synergy foundation token to echeart.
<br/>

<b>Light Theme</b>

```
<div id="eBarChart" class="echart echarts-light"></div>
```

<b>Dark Theme</b>
```
<div id="eBarChart" class="echart echarts-dark"></div>
```


## ECharts Theme Builder.

You can customize your color code in "ECharts Theme Builder".<br/>
[ECharts Theme Builder](https://echarts.apache.org/en/theme-builder.html)<br/><br/>
