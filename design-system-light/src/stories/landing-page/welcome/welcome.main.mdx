To foster a collaborative design environment that unifies diverse teams, creating a seamless user experience across all products.

## Value
<b>Collaboration</b>: Emphasizing teamwork and shared goals.<br/>
<b>Consistency</b>: Delivering a cohesive look and feel across all platforms.<br/>
<b>Innovation</b>: Encouraging creative solutions through collective input.

## Principles
<b>User-Centric Design</b>: Prioritizing user needs in every design decision.<br/>
<b>Data-Driven Approach</b>: Utilizing data insights to guide design choices.<br/>
<b>Continuous Improvement</b>: Regularly refining designs based on feedback and emerging trends.<br/>

# Installation Guide

## Prerequisites

Before you begin, ensure you have:
- A Synopsys account
- npm installed on your system
- Appropriate access permissions

## Installation Steps

### Step 0: Request Access
Email **Vijayabhaskar Katta** (kattav@synopsys.com) and CC dc-uiux-support@synopsys.com with the following details:


<div className="register-form">
- Repositry: dc-npm
- Group: dc-core-team
- Username: [Your Name]
- Email: [Your Company Email]
</div>

### Step 1: Access JFrog Artifactory
1. Navigate to JFrog Artifactory: https://us01-artifactory.internal.synopsys.com/ui/repos/tree/General/dc-npm
2. Log in using your Synopsys account
3. Select the **dc-npm** repository

### Step 2: Generate Token
1. Click **Set me Up** in the upper right corner
2. Navigate to the Configure Tab
3. Enter your password
4. Click "Generate Token & Create Instructions"

<img src={Step2} alt="Jfrog page" />
<img src={Step22} alt="Jfrog page" />

### Step 3: Configure .npmrc
1. Copy the **Edit.npmrc(Scoped)** content
2. Set scope as **@dc**
3. Create or update your .npmrc file:
   - For Mac users: Located in your USER root folder
   - For Windows users: Located at %USERPROFILE%/.npmrc
   - Paste the copied content into this file

**Note**: If you encounter npm configuration errors, please run <b>npm config fix</b>.<br/>
**Note**: After initial configuration, you won't need to repeat this process for future downloads.<br/>

<img src={Step3} alt="Jfrog page" />

### Step 4: Install Dependencies
Run the following command:
```bash
npm i @dc/design-system-core
```

### Step 5: Verify Installation
Check your package.json file. You should see @dc/design-system-core in the dependencies:
```json
{
    "dependencies": {
        "@dc/design-system-core": "[installed version]"
    }
}
```

#### Version Management
- By default, the latest version will be installed
- To install a specific version, use:

  ```bash
  npm install @dc/design-system-core [desired_version]
  ```
