1. 위치 (Position) 테스트

12방향 `position` 속성과 `arrow` 속성을 테스트합니다. 버튼을 클릭하여 확인하세요.

2. 주요 속성 (Props) 및 슬롯 (Slots) 테스트

화살표 없음 (arrow="false")

No Arrow 화살표가 없습니다. 이 Popconfirm은 arrow 속성이 없습니다.

버튼 텍스트 커스텀 (confirm-text, cancel-text)

Custom Text 버튼 텍스트 변경 버튼이 '동의'와 '거절'로 표시됩니다.

바깥 클릭으로 닫기 비활성화 (closable="false")

Not Closable 닫기 비활성화 이 Popconfirm은 바깥 영역을 클릭해도 닫히지 않습니다.

모든 슬롯 사용 (icon, title, default)

All Slots 정말로 삭제하시겠습니까? 이 작업은 되돌릴 수 없습니다. 신중하게 결정해주세요.

3. 프로그래밍 방식 제어 (Methods) 테스트

trigger="none"으로 설정하고, 외부 버튼으로 show(), hide() 메서드를 호출합니다.

Show Popconfirm Hide Popconfirm
프로그래밍 제어 외부 버튼으로 제어됩니다.

4. 비활성화 (Disabled) 상태 테스트

Popconfirm 자체 비활성화 (disabled="true")

Component Disabled 열리지 않아야 합니다.

부모 요소(버튼) 비활성화

Parent Disabled 열리지 않아야 합니다.
