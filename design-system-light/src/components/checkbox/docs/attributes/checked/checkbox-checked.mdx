# Checked
## Definition
Use the checked attribute to activate the checkbox. <br/>
When a checkbox is `checked`, it displays a checkmark inside the checkbox. 

## Boolean Attribute Handling
In the source code, checked = "" should be modified as follows
- Set `checked` attribute and don't put a value after it.
```
<sy-checkbox checked></sy-checkbox>
```
- Alternatively, you can explicitly set true but to not apply `checked`, remove the attribute instead of setting its value to false.

```
<sy-checkbox checked="true"></sy-checkbox>
```
