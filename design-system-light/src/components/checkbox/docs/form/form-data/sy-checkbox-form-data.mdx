# Form Integration / FormData

`sy-checkbox` is a **form-associated custom element**. It plugs into any plain HTML
`<form>` across frameworks (React, Vue, Angular, vanilla) with `FormData`, `required`,
`reset`, and custom validity all working natively:

- Submits its `value` (defaults to `"on"`) under the given `name` when `checked`.
- Honours `required` via native constraint-validation.
- Responds to `<form>.reset()` (via `formResetCallback`) and disabled fieldsets
  (via `formDisabledCallback`).
- Supports state restoration on back/forward navigation (`formStateRestoreCallback`).

```html
<form>
  <sy-checkbox name="newsletter" value="yes">Subscribe</sy-checkbox>
  <sy-checkbox name="agree" value="accepted" required>
    I accept
    <div slot="error"><p>Required</p></div>
  </sy-checkbox>
</form>
```
