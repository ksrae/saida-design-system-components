# Selected Event
## Definition
This event is triggered when user clicks the step.<br/>
Use <b>event.detail.index</b> to get index of the selected step.<br/>
Also, use <b>event.detail.step</b> to get information of the selected step element.<br/>
Property <b>clickable</b> attributes must be set in sy-steps tags.
