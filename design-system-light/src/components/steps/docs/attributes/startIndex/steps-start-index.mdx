# Start Index
## Definition
By setting startIndex to change starting index of a step component. <br/>
Useful for resuming processes or dynamically adjusting the starting step based on user actions.<br/>
Be sure to add an offset to currentIndex as well, unless the currentIndex will be set to startIndex.
