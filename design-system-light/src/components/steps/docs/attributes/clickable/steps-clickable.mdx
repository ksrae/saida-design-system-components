# Clickable
## Definition
Determines whether steps are clickable, allowing users to navigate to the step directly by clicking on step. 


## Boolean Attribute Handling
In the source code, <b>clickable = ""</b> should be modified as follows
- Set `clickable` attribute and don't put a value after it.
```
<sy-steps clickable></sy-steps>
```
- Alternatively, you can explicitly set it to true, but to not to applied `clickable`, remove the attribute instead of setting its value to false.

```
<sy-steps clickable="true"></sy-steps>
```
