# Complete
## Definition
Determines all steps are done.


## Boolean Attribute Handling
In the source code, <b>complete = ""</b> should be modified as follows
- Set `complete` attribute and don't put a value after it.

```
<sy-steps complete></sy-steps>
```
- Alternatively, you can explicitly set it to true, but to not apply `complete`, remove the attribute instead of setting its value to false.

```
<sy-steps complete="true"></sy-steps>
```
