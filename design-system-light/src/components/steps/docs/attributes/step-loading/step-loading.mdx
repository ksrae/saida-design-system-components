# Loading
## Definition
Shows loading on the step.<br/>
While loading attribute sets, loader is visible.<br/>


## Boolean Attribute Handling
In the source code, <b>loading = ""</b> should be modified as follows
- Set `loading` attribute and don't put a value after it.
```
<sy-step loading></sy-step>
```
- Alternatively, you can explicitly set it to true, but to not to applied `loading`, remove the attribute instead of setting its value to false.

```
<sy-step loading="true"></sy-step>
```
