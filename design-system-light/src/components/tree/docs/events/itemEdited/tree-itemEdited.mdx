# Editable Item
## Definition
The tree node can be edited by setting the `editable` attribute. <br/>
When the editable attribute is set, the user can add a new node or edit, remove the selected node. <br/>


## itemAdded
When clicking the + button and input and add cancle button is shown bottom of the node.<br/>
itemAdded event is triggered, if clicking the add button.<br/>
This event includes the value of the parent node, the label, value, and level (or depth) of the newly added node. <br/>
The value is formed as "(parent's value)-(Date.now())" to get uniqueness.<br/>

## itemEdited
When blur event is triggered from input element after edting item label, the itemEdited event is triggered. <br/>
This event includes the value, and label of the node. <br/>

## itemRemoved
When clicking the - button, the itemRemoved event is triggered. <br/>
The node along with all its child nodes are removed. <br/>
This event includes the value, and label of the node.
