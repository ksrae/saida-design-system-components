# ItemChecked
## Definition
When a node is checked or unchecked, the itemChecked event is triggered. <br/>
This event includes the value, label, and checked of the node. <br/>
To fire this event, checkable attribute must be set.

To get the checked state of all nodes, use `nodesChanged` event. <br/>
