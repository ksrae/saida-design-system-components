# Expandable
## Definition
If the node has child nodes, the node is able to expand or collapse.<br/>
It is possible to set expanded as default by modifying the expanded value of each node.

```json
{
	label: 'item1',
	value: '1', 
	expanded: false, // set true to allow be expanded when expandable attribute is set. default is false
}
```


## Boolean Attribute Handling
In the source code, <b>expandable = ""</b> should be modified as follows
- Set `expandable` attribute and don't put a value after it.
```
<sy-tree expandable></sy-tree>
```
- Alternatively, you can explicitly set it to true, but to not apply `expandable`, remove the attribute instead of setting its value to false.

```
<sy-tree expandable="true"></sy-tree>
```
