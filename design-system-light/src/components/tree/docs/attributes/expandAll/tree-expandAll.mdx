# expandAll
## Definition
All nodes are expanded when the tree is visible. <br/>
It is possible to expand or collapse all nodes at once, by modifying this attribute value in script codes.


## Boolean Attribute Handling
In the source code, <b>expandAll = ""</b> should be modified as follows
- Set `expandAll` attribute and don't put a value after it.
```
<sy-tree expandAll></sy-tree>
```
- Alternatively, you can explicitly set it to true, but to not to applied `expandAll`, remove the attribute instead of setting its value to false.

```
<sy-tree expandAll="true"></sy-tree>
```
