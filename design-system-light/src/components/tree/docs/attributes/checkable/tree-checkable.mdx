# Checkable
## Definition
A checkbox shows in front of each tree item. <br/>
The result of the check affects both parent and child nodes. <br/>
Whenever a node is checked or unchecked, itemChecked event is triggered. <br/><br/>


## Boolean Attribute Handling
In the source code, <b>checkable = ""</b> should be modified as follows
- Set `checkable` attribute and don't put a value after it.
```
<sy-tree checkable></sy-tree>
```
- Alternatively, you can explicitly set it to true, but to not apply `checkable`, remove the attribute instead of setting its value to false.

```
<sy-tree checkable="true"></sy-tree>
```
