# Draggable
## Definition
All tree items can be moved by dragging. <br/>
You can drop above and below the node, and when dropped on a node, it is added as a child of that node. <br/>
However, it is prohibited to drop on itself or its child nodes. <br/>
The UI indicates the drop zones.


## Boolean Attribute Handling
In the source code, <b>draggable = ""</b> should be modified as follows
- Set `draggable` attribute and don't put a value after it.
```
<sy-tree draggable></sy-tree>
```
- Alternatively, you can explicitly set it to true, but to not to apply `draggable`, remove the attribute instead of setting its value to false.

```
<sy-tree draggable="true"></sy-tree>
```
