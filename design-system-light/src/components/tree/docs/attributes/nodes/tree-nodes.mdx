# Nodes
## Definition
Apply the nodes that make up the tree. The nodes value must be in JSON format that matches the TreeNode interface.<br/><br/>

To apply the JSON directly to the nodes property, you need to inject the value in the script.<br/>


```html
<sy-tree id="tree1"></sy-tree>

<script>
    const nodes = [
        {
          label: 'parent 1', 
          value: '100', 
          children: [

          ]
        }
      ];

    const tree = document.querySelector('sy-tree#tree1');
    tree.nodes = nodes;   
    </script>
```
