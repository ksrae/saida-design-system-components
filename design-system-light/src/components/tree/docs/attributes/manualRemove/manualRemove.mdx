# ManualRemove
## Definition
It `manualRemove` attribute is set, the tree item does not be removed by clicking the `-` icon.<br/>
This is an attribute that allows the user to confirm the action once more through a confirmation modal when the user removes a tree node.<br/><br/>
To remove manually, use `removeNode` method. The method requires `value` as an argument.<br/>


## Boolean Attribute Handling
In the source code, <b>manualRemove = ""</b> should be modified as follows
- Set `manualRemove` attribute and don't put a value after it.
```
<sy-tree manualRemove></sy-tree>
```
- Alternatively, you can explicitly set it to true, but to not apply `manualRemove`, remove the attribute instead of setting its value to false.

```
<sy-tree manualRemove="true"></sy-tree>
```
