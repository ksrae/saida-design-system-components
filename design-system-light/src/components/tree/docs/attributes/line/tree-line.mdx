# Line
## Definition
Draws lines between tree items.


## Boolean Attribute Handling
In the source code, <b>line = ""</b> should be modified as follows
- Set `line` attribute and don't put a value after it.
```
<sy-tree line></sy-tree>
```
- Alternatively, you can explicitly set it to true, but to not apply `line`, remove the attribute instead of setting its value to false.

```
<sy-tree line="true"></sy-tree>
```
