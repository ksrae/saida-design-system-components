# Select
## Definition
Select element allows users to choose multiple items by setting the <b>mode</b>  to `multiple`.<br/>
If the <b>mode</b> attribute is set to `multiple`, selected items can be removed by clicking the item in the select.<br/>
Select also provides a <b>clear all</b> button by adding <b>clearable</b> attribute.<br/>

# Select Option
## Definition
Each <b>sy-option</b> within the <b>sy-select</b> represents a selectable item. <br/>
Options are typically presented as text labels with label attributes; however, they can also include additional elements such as icons or checkboxes for multi-select mode using slotContent.
