# Select

A selection control that lets a user pick one or more options from a dropdown list. Supports `default`, `searchable`, `multiple`, and `tag` interaction modes.

## Usage

```html
<sy-select placeholder="Choose" mode="default">
  <sy-select-option value="apple" label="Apple">Apple</sy-select-option>
  <sy-select-option value="banana" label="Banana">Banana</sy-select-option>
</sy-select>
```
