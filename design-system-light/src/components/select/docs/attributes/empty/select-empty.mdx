# Empty
## Definition
Showing empty icon when opening the select.
Any options are not allowed to added when the `empty` attribute is set.<br/>
It indicates to the user that no selections can be made in an empty state, where the select list has no available choices.

## Boolean Attribute Handling
In the source code, <b>empty = ""</b> should be modified as follows
- Set `empty` attribute and don't put a value after it.
```
<sy-select empty></sy-select>
```

- Alternatively, you can explicitly set it to true, but to not apply `empty`, remove the attribute instead of setting its value to false.
```
<sy-select empty="true"></sy-select>
```
