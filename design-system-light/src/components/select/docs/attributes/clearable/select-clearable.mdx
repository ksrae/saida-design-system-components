# Clearable
## Definition
This attribute provides buttons for clear all selected option at once.<br/>
At least one option must be selected to activate clearable.


## Boolean Attribute Handling
In the source code, <b>clearable = ""</b> should be modified as follows
- Set `clearable` attribute and don't put a value after it.
```
<sy-select clearable></sy-select>
```
- Alternatively, you can explicitly set it to true, but to not apply `clearable`, remove the attribute instaed of setting its value to false.

```
<sy-select clearable="true"></sy-select>
```
