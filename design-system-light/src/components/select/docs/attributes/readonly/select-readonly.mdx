# Readonly
## Definition
The `readonly` attribute is used to make the select component non-interactive, meaning users cannot change its value through the user interface. <br/>
This is useful when you want to display a value without allowing the user to modify it.

## Boolean Attribute Handling
In the source code, <b>readonly = ""</b> should be modified as follows
- Set `readonly` attribute and don't put a value after it.
```
<sy-select readonly></sy-select>
```
- Alternatively, you can explicitly set it  to true, but to not apply the `readonly`, remove the attribute instead of setting its value to false.

```
<sy-select readonly="true"></sy-select>
```
