# Error
## Definition
The border of select is set to the <b>ERROR</b> status.

## Boolean Attribute Handling
In the source code, <b>error = ""</b> should be modified as follows
- Set `error` attribute and don't put a value after it.
```
<sy-select error></sy-select>
```
- Alternatively, you can explicitly set it to true, but to not apply `error`, remove the attribute instead of setting its value to false.

```
<sy-select error="true"></sy-select>
```
