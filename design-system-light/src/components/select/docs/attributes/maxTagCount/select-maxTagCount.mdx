# MaxTagCount
## Definition
When the `mode` attribute is set to `multiple`, <b>maxTagCount</b> attribute decides how many selected values are exposed. <br/>
If more items are selected than the <b>maxTagCount</b>, the additional selected numbers are displayed as a list on the right side of the select. <br/>
Only number type value is acceptable. Setting <b>0</b> has the same effect as not defining <b>maxTagCount</b>.
