# Option ShowTooltip
## Definition
The `showTooltip` attribute is used to define visibility of the placeholder(tooltip) for the select option.

## Boolean Attribute Handling
In the source code, <b>showTooltip = ""</b> should be modified as follows
- Set `showTooltip` attribute and don't put a value after it.
```
<sy-select showTooltip></sy-select>
```
- Alternatively, you can explicitly set it to true, but to not apply `showTooltip`, remove the attribute instead of setting its value to false.

```
<sy-select showTooltip="true"></sy-select>
```
