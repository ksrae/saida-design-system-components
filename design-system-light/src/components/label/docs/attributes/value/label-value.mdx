# Value
## Definition
The `value` attribute of a label is used to specify the value of the label when it is associated with a form control. This is particularly useful for input elements where you want to provide a default value or a value that can be submitted with the form.

## Position
Specifies that the label should be aligned to the `left` or `right`.
