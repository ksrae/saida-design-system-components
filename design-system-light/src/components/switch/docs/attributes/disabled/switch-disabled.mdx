# Disabled
## Definition
To set the switch disabled

## Boolean Attribute Handling
In the source code, <b>disabled = ""</b> should be modified as follows
- Set `disabled` attribute and don't put a value after it.
```
<sy-switch disabled></sy-switch>
```
- Alternatively, you can explicitly set it  to true, but to not apply the `disabled`, remove the attribute instead of setting its value to falsㄷ.

```
<sy-switch disabled="true"></sy-switch>
```
