# Readonly
## Definition
To set the switch readonly state 
It makes the switch not mutable, meaning the user can not edit the control.<br/>

When read-only mode is activated, the click event is not applied. <br/>

## Boolean Attribute Handling
In the source code, <b>readonly = ""</b> should be modified as follows
- Set `readonly` attribute and don't put a value after it.
```
<sy-switch readonly></sy-switch>
```
- Alternatively, you can explicitly set it to true but to not apply `readonly`, remove the attribute instead of setting its value to false.

```
<sy-switch readonly="true"></sy-switch>
```
