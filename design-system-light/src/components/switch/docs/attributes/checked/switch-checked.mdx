# Checked
## Definition
To set the switch checked

## Boolean Attribute Handling
In the source code, <b>checked = ""</b> should be modified as follows
- Set `checked` attribute and don't put a value after it.
```
<sy-switch checked></sy-switch>
```
- Alternatively, you can explicitly set it to true, but to not apply `checked`, remove the attribute instead of setting its value to false.

```
<sy-switch checked="true"></sy-switch>
```
