# ShowText
## Definition
The `showText` attribute allows the `colorpicker` to display the selected color value as text alongside the color selection interface. <br/>
This can be useful for users who prefer to see the color value in a readable format, such as hex or RGB values.

## Boolean Attribute Handling
In the source code, showText = "" should be modified as follows. <br/>
Set the <b>showText</b> attribute without assigning a value to it. <br/>
```
<sy-colorpicker showText></sy-colorpicker>
```
Alternatively, you can explicitly set it to true, but to not apply `showText`, just remove the attribute instead of setting its value to false.

```
<sy-colorpicker showText="true"></sy-colorpicker>
```
