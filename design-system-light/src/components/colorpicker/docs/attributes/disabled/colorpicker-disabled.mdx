# Disabled
## Definition
Use the disabled attribute to disable a `colorpicker`. <br/>
If disabled attribute is set, any events will be prevented.

## Boolean Attribute Handling
In the source code, disabled = "" should be modified as follows. <br/>
Set the <b>disabled</b> attribute without assigning a value to it. <br/>
```
<sy-colorpicker disabled></sy-colorpicker>
```
Alternatively, you can explicitly set it to true, but to not apply `disabled`, just remove the attribute instead of setting its value to false.

```
<sy-colorpicker disabled="true"></sy-colorpicker>
```
