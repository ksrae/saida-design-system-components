# Inline
## Definition
The `inline` attribute allows the `colorpicker` to be displayed inline with other content, rather than as a separate popup or modal. <br/>

## Boolean Attribute Handling
In the source code, inline = "" should be modified as follows. <br/>
Set the <b>inline</b> attribute without assigning a value to it. <br/>
```
<sy-colorpicker inline></sy-colorpicker>
```
Alternatively, you can explicitly set it to true, but to not apply `inline`, just remove the attribute instead of setting its value to false.

```
<sy-colorpicker inline="true"></sy-colorpicker>
```
