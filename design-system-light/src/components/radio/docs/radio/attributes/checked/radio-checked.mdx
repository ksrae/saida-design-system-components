# Checked
## Definition
The checked state of the radio item.

## Boolean Attribute Handling
In the source code, <b>checked = ""</b> should be modified as follows
- Set `checked` attribute and don't put a value after it.
```
<sy-radio checked></sy-radio>
```
- Alternatively, you can explicitly set it to true, but to not to apply `checked`, remove the attribute instead of setting its value to false.

```
<sy-radio checked="true"></sy-radio>
```
