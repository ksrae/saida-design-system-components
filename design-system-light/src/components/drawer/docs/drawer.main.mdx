# Drawer
## Definition
A drawer is that slides in from the side of the screen, typically used to provide additional options. <br/>
It allows users to access various features or sections of an application without cluttering the main interface. <br/>
The drawer can be opened or closed through user interaction, such as swiping or tapping a button.
