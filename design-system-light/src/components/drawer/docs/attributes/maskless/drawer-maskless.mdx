# Maskless
## Definition
A mask is not created. <br/>
If a mask is not generated, events such as clicks on the DOM outside the drawer area can occur 
while the drawer is open, which may lead to unexpected behaviors.


## Boolean Attribute Handling
In the source code, maskless = "" should be modified as follows
- Set `maskless` attribute and don't put a value after it.
```
<sy-drawer maskless></sy-drawer>
```
- Alternatively, you can explicitly set it to true but to not apply `maskless`, remove the attribute instead of setting its value to false.
```
<sy-drawer maskless="true"></sy-drawer>
```
