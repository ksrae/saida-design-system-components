# setFocus() / setBlur()

`setFocus()` and `setBlur()` are public methods that programmatically control the focus state of the autocomplete.

The example below auto-focuses the input after 1 second and then blurs it after 4 seconds.

```js
const elem = document.querySelector('sy-autocomplete');
elem.setFocus();
elem.setBlur();
```
