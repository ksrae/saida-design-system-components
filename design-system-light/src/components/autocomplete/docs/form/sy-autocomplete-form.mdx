# Form Integration

`sy-autocomplete` is a **form-associated custom element** (`ElementInternals`-based). It works
inside any plain HTML `<form>` regardless of the surrounding frontend framework:

- Submits its `value` under the given `name` — picked up by `FormData`.
- Honours `required`, and participates in native `reportValidity()` / constraint-validation.
- Responds to `<form>.reset()` (via `formResetCallback`) and disabled fieldsets (via
  `formDisabledCallback`).
- Supports state restoration on back/forward navigation (`formStateRestoreCallback`).
