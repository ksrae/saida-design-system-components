# Case Sensitive

## Definition
Auto Complete provides case-sensitive when searching for data.<br/>Set <b>caseSensitive</b> property for case-sensitive while searching; otherwise, it searches without distinction.<br/><br/>
Type "a" in each test case. <br/>"ABC" is listed on "Default" case; however, no data is shown on "Case Sensitive" case.

## Boolean Attribute Handling
In the source code, caseSensitive = "" should be modified as follows
- Set "caseSensitive" attribute and don't put a value after it.
```
<sy-autocomplete caseSensitive></sy-autocomplete>
```
- Alternatively, you can explicitly put true but to not to applied "caseSensitive" remove the attribute not setting value to false.

```
<sy-autocomplete caseSensitive="true"></sy-autocomplete>
```
