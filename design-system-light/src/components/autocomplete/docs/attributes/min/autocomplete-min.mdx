# Min

## Definition
Minimum number of characters to trigger suggestions. <br/><br/>
If a user inputs more characters than min number, only suggested items are shown on the list.<br/>
If a user inputs less characters than min number, all items are shown on the list.<br/><br/>
Default value is 0.
