# Selected Event
## Definition
Selecting any suggested item emits <b>selected</b> event.<br/><br/>
event.detail includes the value of the selected item and the validation status of the autocomplete component.

```
event: {
  detail: {
    value: string; // The value of the selected item
    isValid: boolean; // Indicates if the selection is valid
    status: 'valueMissing' | 'custom' | ''; // The validation status of the autocomplete component
  }
}
```
