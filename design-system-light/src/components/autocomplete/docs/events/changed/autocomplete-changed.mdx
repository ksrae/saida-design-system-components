# Changed Event
## Definition
The `changed` event is triggered when the value of the autocomplete input changes, either through user input or programmatically.<br/> 
This event provides details about the current value and its validation status.

```
event: {
  detail: {
    value: string; // The value of the selected item
    isValid: boolean; // Indicates if the selection is valid
    status: 'valueMissing' | 'custom' | ''; // The validation status of the autocomplete component
  }
}
```
