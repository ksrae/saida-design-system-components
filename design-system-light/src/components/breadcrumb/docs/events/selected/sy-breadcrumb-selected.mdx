# selected

The `selected` event bubbles up from a clicked `<sy-breadcrumb-item>`. The event's `detail.target` is the clicked item.

```js
document.querySelector('sy-breadcrumb').addEventListener('selected', (e) => {
  console.log(e.detail.target.innerText);
});
```
