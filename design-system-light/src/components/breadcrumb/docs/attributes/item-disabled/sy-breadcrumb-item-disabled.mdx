# Item Disabled

Add the `disabled` attribute to a `<sy-breadcrumb-item>` to make it non-interactive. Disabled items do not emit the `selected` event and are visually muted.

```html
<sy-breadcrumb-item disabled>Unavailable</sy-breadcrumb-item>
```
