# Disabled
## Definition
Sets disabled attribute to an item to disable the item.<br/><br/>

There are two examples to express how disabled attribute works.<br/>
In the example, the second item in the Disabled Item is disabled because it has the disabled attribute, while the other items are not. 


## Boolean Attribute Handling
In the source code, disabled = "" should be modified as follows
- Set `disabled` attribute and don't put a value after it.
```
<sy-breadcrumb-item disabled></sy-breadcrumb-item>
```
- Alternatively, you can explicitly set it  to `true`, but to not apply the `disabled`, remove the attribute instead of setting its value to`false`.

```
<sy-breadcrumb-item disabled="true"></sy-breadcrumb-item>
```
