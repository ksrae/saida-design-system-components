# Icon
## Definition
Icons are symbols that can be usd to represent various options within an application. <br/><br/>
We basically recommend using the size properties provided by the sy-icon component.<br/>
Icons are also able to be modified its color & size with style.<br/><br/>
You can apply icons in two ways on the web.<br/>
And you can use all the SVG icons you want.<br/><br/>
Synergy uses font-awesome icons as its base.<br/>
FontAwsome Classic (Regular, Solid)<br/>
<img src={syIconKit} alt="sy-icon-kit"></img><br/>
<a href="https://fontawesome.com/icons?t=packs" target="_blank"><u>FontAwsome Icons</u></a>


### How to use icons
**First :** Please insert SVG code directly using slotContent from Sy-Icon.<br/>
1.Copy the fontawesome svg icon or other SVG icon code you want.<br/>
2.Paste SVG code inside sy-icon as below image.<br/>
<img src={syIconUse} alt="sy-icon-use" width="70%"></img><br/>
<img src={syIconUse01} alt="sy-icon-use-slot" width="70%"></img><br/>
**Second :** Put the svg icon in your application development folder<br/>
You can use the icon by entering the SVG path in the path of Sy-Icon.<br/><br/>
1.Put the icon you want to use in the folder you want.<br/>
2.Specify the icon path in sy-icon path="url" as shown in the image below.<br/>
<img src={syIconUse02} alt="sy-icon-use-path" width="70%"></img><br/><br/>

### Legacy Icons
For the Legacy application, please download the font-awesome icon and use the SVG icon.<br/>
Sy-icon is not available on non-Web Legacy products.

### Custom icons guide
If you would you like make custom icons.<br/>
Please check out the font-awesome custom icon guide.<br/>
<a href="https://docs.fontawesome.com/web/add-icons/upload-icons/icon-design" target="_blank"><u>Custom icons guide of FontAwesome</u></a>

```style
<!-- Size example (Please make the icon size square) -->
<sy-icon size="medium">
<sy-icon style="width:100px; height:100px;>

<!-- Color example -->
<sy-icon style="color:purple">
<sy-icon style="color:#800080">

```

# Available Icons

All available icons from the library are shown below.<br/>
These icons are resized to the right size when used in conjunction with `<sy-icon>`.<br/><br/>

Click or tap on any icon to copy its raw SVG code to your clipboard. To use the icon, paste the copied code **directly inside an `<sy-icon>` component**.<br/>
Developers can use more diverse icons by using 'path' or by input other SVG. However, such icons cannot guarantee the appropriate type of icon when used in conjunction with `<sy-icon>`.

**Example:**
```html
<sy-icon size="large">
  <!-- attach svg code here -->
  <svg xmlns="http://www.w.org/2000/svg" viewBox="0 0 640 640"><path fill="currentColor" d="..."></path></svg>
</sy-icon>
```
