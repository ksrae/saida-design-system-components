# Type
## Definition
This attribute is used to set the type of skeleton.<br/>


## Values
- `text`: A text skeleton is displayed.
- `avatar`: An avatar skeleton is displayed. If the `rows` attribute is set, text skeletons are displayed as much as number of rows.
- 'image': An image skeleton is displayed. If the `rows` attribute is set, text skeletons are displayed as much as number of rows.
- 'gallary': Multiple image skeletons are displayed. If the `rows` attribute is set, multiple image skeletons are displayed as much as number of rows.
- 'button': A button skeleton is displayed. If the `rows` attribute is set, multiple button skeletons are displayed in a rows.
- 'table': A table skeleton is displayed. If the `rows` attribute is set, multiple table rows skeletons are displayed.
