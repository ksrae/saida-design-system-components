# Collapse
## Definition
Collapse allows sections of content to be hidden from view. <br/>
This functionality helps streamline the interface, making it less cluttered and allowing users to focus on the most relevant information.  <br/>
When a section is collapsed, it shows only a title, and users can expand it by clicking or tapping to reveal the panel which has additional information. <br/>
