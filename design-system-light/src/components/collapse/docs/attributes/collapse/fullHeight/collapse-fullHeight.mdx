# Full height
## Definition
Set the `fullheight` attribute on the collapse to remove the background color from all panels.<br/>
This provides a cleaner, less intrusive design by using a 'ghost' style with no background.

## Boolean Attribute Handling
In the source code, <b>fullheight = ""</b> should be modified as follows
- Set `fullheight` attribute and don't put a value after it.
```
<sy-collapse fullheight></sy-collapse>
```

- Alternatively, you can explicitly set it  to true, but to not apply the `fullheight`, remove the attribute instead of setting its value to false.
```
<sy-collapse fullheight="true"></sy-collapse>
```
