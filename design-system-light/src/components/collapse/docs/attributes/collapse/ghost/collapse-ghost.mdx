# Ghost
## Definition
Set the `ghost` attribute on the collapse to remove the background color from all panels.<br/>
This provides a cleaner, less intrusive design by using a 'ghost' style with no background.

## Boolean Attribute Handling
In the source code, <b>ghost = ""</b> should be modified as follows
- Set `ghost` attribute and don't put a value after it.
```
<sy-collapse ghost></sy-collapse>
```

- Alternatively, you can explicitly set it  to `true`, but to not apply the `ghost`, remove the attribute instead of setting its value to `false`.
```
<sy-collapse ghost="true"></sy-collapse>
```
