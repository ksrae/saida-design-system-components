# Borderless
## Definition
By default, the collapse has a border. Sets `borderless` attribute to remove border. <br/>
The borderless aims to provide a more minimalist and modern look. It removes the borders around panels to create a cleaner and less cluttered interface. 

## Boolean Attribute Handling
In the source code, <b>borderless = ""</b> should be modified as follows
- Set `borderless` attribute and don't put a value after it.
```
<sy-collapse borderless></sy-collapse>
```

- Alternatively, you can explicitly set it to true, but to not apply `borderless`, remove the attribute instead of setting its value to false.
```
<sy-collapse borderless="true"></sy-collapse>
```
