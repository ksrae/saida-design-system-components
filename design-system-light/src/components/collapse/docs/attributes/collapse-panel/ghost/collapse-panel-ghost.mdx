# Panel Ghost
## Definition
Sets ghost attribute to the collapse panel to remove its background color. <br/>


## Boolean Attribute Handling
In the source code, ghost = "" should be modified as follows
- Set `ghost` attribute and don't put a value after it.
```
<sy-collapse-panel ghost></sy-collapse-panel>
```

- Alternatively, you can explicitly set it to true, but to not to apply `ghost`, remove the attribute instead of setting its value to false.
```
<sy-collapse-panel ghost="true"></sy-collapse-panel>
```
