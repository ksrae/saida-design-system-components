# openAll() / closeAll() / openPanel(i) / closePanel(i)

Public methods to programmatically control panel state.

```js
const elem = document.querySelector('sy-collapse');
elem.openAll();
elem.closeAll();
elem.openPanel(1);
elem.closePanel(1);
```
