# Min
## Definition
This specifies the minimum number of characters required for the input to be considered valid. <br/>
It is possible to input a value less than the minimum number, which will set the valid status to false.<br/>
While valid is false, the border will change on blur event.<br/>
Default value is `Number.MIN_SAFE_INTEGER`.
