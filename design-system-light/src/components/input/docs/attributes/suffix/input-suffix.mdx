# Suffix
## Definition
To set the suffix icon with slot, add `slot="suffix"` attribute on the tag. <br/>
This is useful to add suffix icon on the input field.<br/>
However, If the input type is <b>password</b> or <b>search</b>, default icon will be set instead of suffix.
