# Navigation Menu
## Definition
A navigation menu provides a list of options or links to help users navigate through an application. <br/>
It appears as a vertical bar containing clickable links to various sections, pages, or features of the application.<br/>

Our Nav element has the following features:

- Provides several tags: `nav-sub`, `nav-item`, `nav-group`.
  - `nav-item`: Displays a value defined by the user. It emits a selected event when clicked.
  - `nav-sub`: Opens a new nav-box and displays the nav-items that belong to it.
  - `nav-group`: Displays the nav-items that belong to the nav-group as children in the nav-box.

- State 
  - `Enabled`: Default state.
  - `Hover`: Visual feedback when users move their cursor over an item.
  - `Active`: Highlighted appearance indicating the current selection.
  - `Focused`: Visual indication when navigating via keyboard.
  - `Disabled`: Visual state for items that are temporarily unavailable for selection. (Refer to the Attributes section for more details)
