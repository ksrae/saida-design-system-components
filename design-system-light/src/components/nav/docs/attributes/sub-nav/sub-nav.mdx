# Sub Nav
## Definition
Sub navigation menu may contains navigation menu items and navigation menu groups as well as other sub navs.<br/>
To apply a sub nav, put all <b> \<sy-nav-item\> </b>  tags into <b> \<sy-nav-sub\>. </b>
<br/>
The title of sub nav must be configured with the title attribute.
