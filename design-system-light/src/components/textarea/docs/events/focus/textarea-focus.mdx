# Focus & Blur Function
## Definition of Focus
Using `setFocus()` function to set focus to the textarea manually.<br/>
When `setFocus()` is called, input field has focus.<br/>

## Definition of Blur
Using `setBlur()` function to set blur to the textarea manually.<br/>
When `setBlur()` is called, input field has lost its focus.<br/>

In the example, textarea get a focus in 1 second and blur after 4 seconds.<br/>
To test this clearly, this page must be activated.
