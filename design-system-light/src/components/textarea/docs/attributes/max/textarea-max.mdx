# Max
## Definition
Setting maximum text length for the textarea. <br/>
Text length of the textarea cannot be greater than the maximum value. <br/>
Maximum value is set to the minimum value if the maximum value is less than the minimum value. <br/>
Maximum value must be set to expose the counter.
