# Required
## Definition
The required indicator, also known as a mandatory indicator, is a visual cue used to denote that a particular input within a form is mandatory or required. 
The validation for the `required` attribute only works when the `textarea` is inside a form.


## Boolean Attribute Handling
In the source code, <b>required = ""</b> should be modified as follows
- Set `required` attribute and don't put a value after it.
```
<sy-textarea required></sy-textarea>
```
- Alternatively, you can explicitly set it to true, but to not apply `required` attribute, remove it instead of setting its value to false.
```
<sy-textarea required="true"></sy-textarea>
```
