# Clearable
## Definition
It adds a clear button within the input field, allowing users to quickly clear the entered text with a single click.


## Boolean Attribute Handling
In the source code, <b>clearable = ""</b> should be modified as follows
- Set `clearable` attribute and don't put a value after it.
```html
<sy-textarea clearable></sy-textarea>
```

- Alternatively, you can explicitly set `clearable` to true, but to not apply `clearable`, remove the attribute instead of setting its value to false.
```html
<sy-textarea clearable="true"></sy-textarea>
```
