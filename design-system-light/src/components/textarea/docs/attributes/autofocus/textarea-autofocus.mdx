# Autofocus
## Definition
Set focus to the textarea when page loaded.<br/>
It may not work if other dom is focused.


## Boolean Attribute Handling
In the source code, <b>autofocus = ""</b> should be modified as follows
- Set `autofocus` attribute and don't put a value after it.
```
<sy-textarea autofocus></sy-textarea>
```
- Alternatively, you can explicitly set it to true, but to not apply `autofocus`, remove the attribute instead of setting its value to false.

```
<sy-textarea autofocus="true"></sy-textarea>
```
