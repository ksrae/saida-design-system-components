# Card
## Definition
Cards can be used to group related subjects in a container.

## Basic Card
Basic cards aren’t very exciting, but they can display any content you want them to.<br/>
The slot without designated name will be displayed as a card body.

Footers can be used to display actions, summaries, or other relevant content.

## Collapsible Card
For lengthy content, you can use a collapsible card.
To enable this feature, set the <b>collapsible</b> property to true. <br/>
Note: A header slot must be provided to show the collapse/expand button.
