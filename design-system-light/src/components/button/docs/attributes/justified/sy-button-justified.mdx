# Full Width

The `fullWidth` attribute (HTML attribute: `full-width`) forces the button to expand to the
width of its parent container. Use this for justified layouts (e.g., action rows, form footers).

The legacy attribute `justified` is accepted for backward compatibility, but new markup should
use `full-width` per the spec.
