# Loading
## Definition
To set whether the button shows loading state or doesn't.<br/>
If the value of `loading` is true, it behaves according to the following conditions

- If `<sy-spinner>` is loaded, it shows the spinner.
- If `<sy-spinner>` is not loaded, it will show the 'Loading...' message.


## Boolean Attribute Handling
In the source code, loading = "" should be modified as follows
- Set `loading` attribute and don't put a value after it.
```
<sy-button loading></sy-button>
```
- Alternatively, you can explicitly set true but to not apply `loading` attribute, remove the it instead of setting value to false.

```
<sy-button loading="true"></sy-button>
```
