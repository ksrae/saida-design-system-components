# Type
The `type` attribute of the button determines its behavior. It can be set to one of the following values: <br/>


- `button`: The default type, which does not submit a form. Most commonly used for general actions. <br/> 
- `submit`: Submits the form data to the server.<br/>
- `reset`: Resets the form data to its initial values.
