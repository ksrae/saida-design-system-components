# Segment
## Definition
The segment used to display multiple values in different colors in the progress circular. <br/>
The value should be an array of objects with the following properties:
- `percent`: The percentage of the segment. The value should be between 0 and 100.
- `color`: The color of the segment. The value should be a valid CSS color.

<br/>
The percent of the last segment must have a value less than or equal to 100 or number of percent property, if exists. <br/>
If the percent of the last segment is less than 100 (or percent property), the remaining segment will be filled with color which is set on the color property. <br/>
