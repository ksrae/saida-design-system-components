# HideonSingle
## Definition
If `hideonSingle` is set to true, pagination is hidden when total page count is <b>one (or less)</b>.

## Boolean Attribute Handling
In the source code, <b>hideonSingle = ""</b> should be modified as follows
- Set the `hideonSingle` attribute and don't put a value after it.
```
<sy-pagination hideonSingle></sy-pagination>
```

- Alternatively, you can explicitly set it to true, but to not apply `hideonSingle`, remove the attribute instead of setting its value to false.
```
<sy-pagination hideonSingle="true"></sy-pagination>
```
