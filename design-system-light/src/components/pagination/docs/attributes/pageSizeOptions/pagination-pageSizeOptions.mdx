# Page Size Options
## Definition
To specify the page size changer options.<br/><br/>
The total number of pages changes according to the page size options.<br/>
(e.g. if the page size options change from <b>10 to 20</b>, the total number of pages will change from <b>30 to 15</b>.)<br/><br/>
If the current active page is greater than the total number of pages after the page size option has changed, the active page will be set to the last page.
