# MaxTagCount
## Definition
The maximum number of tags to display.<br/>
If the number of tags exceeds this value, the remaining tags will be hidden and replaced with a "+X" tag, where X is the number of hidden tags.<br/><br/>
Checkable property is required to use this attribute.<br/>
