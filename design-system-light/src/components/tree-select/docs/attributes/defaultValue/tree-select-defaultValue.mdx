# Default Value
## Definition
The `defaultValue` attribute is used to set the default value of the tree select.<br/>
This attribute is only applied when <b>checkable</b> is not set.

When <b>checkable</b> is set, you can implement the defaultValue feature by setting the checked state of each node.<br/>
The 'defaultValue' must be a string value that matches the value of one of the tree node.
