# Expandable
## Definition
When input any words, expand tree until matched tree-item is found.<br/>

## Boolean Attribute Handling
In the source code, <b>expandable = ""</b> should be modified as follows
- Set `expandable` attribute and don't put a value after it.
```
<sy-tree-select expandable></sy-tree-select>
```
- Alternatively, you can explicitly set it to true, but to not apply `expandable`, remove the attribute instead of setting its value to false.

```
<sy-tree-select expandable="true"></sy-tree-select>
```
