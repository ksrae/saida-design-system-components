# ExpandAll
## Definition
The `expandAll` attribute is a boolean value that determines if all nodes in the tree are expanded or collapsed.

## Boolean Attribute Handling
In the source code, <b>expandAll = ""</b> should be modified as follows
- Set `expandAll` attribute and don't put a value after it.
```
<sy-tree-select expandAll></sy-tree-select>
```
- Alternatively, you can explicitly set it to true, but to not apply `expandAll`, remove the attribute instead of setting its value to false.

```
<sy-tree-select expandAll="true"></sy-tree-select>
```
