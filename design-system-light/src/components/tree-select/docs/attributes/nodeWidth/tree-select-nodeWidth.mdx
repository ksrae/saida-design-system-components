# Node Width
## Definition
Sets max width of all tree select nodes.
When a number is entered as a `nodeWidth`, the max-width is applied in px units. <br/>
If the content of the tree item exceeds the `nodeWidth`, an ellipsis will appear, and the full content of the tree item will be displayed as a tooltip.
