# Loading
## Definition
The `loading` attribute is a boolean that determines if the tree select is in a loading state.

## Boolean Attribute Handling
In the source code, <b>loading = ""</b> should be modified as follows
- Set `loading` attribute and don't put a value after it.
```
<sy-tree-select loading></sy-tree-select>
```
- Alternatively, you can explicitly set it to true, but to not apply `loading`, remove the attribute instead of setting its value to false.

```
<sy-tree-select loading="true"></sy-tree-select>
```
