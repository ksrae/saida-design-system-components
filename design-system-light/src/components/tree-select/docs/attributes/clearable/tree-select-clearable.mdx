# Clearable
## Definition
A `clearable` attribute allows users to clear the selected value in a tree select component, enhancing usability by providing a quick way to reset the selection.

## Boolean Attribute Handling
In the source code, <b>clearable = ""</b> should be modified as follows
- Set `clearable` attribute and don't put a value after it.
```
<sy-tree-select clearable></sy-tree-select>
```
- Alternatively, you can explicitly set it to true, but to not apply `clearable`, remove the attribute instead of setting its value to false.

```
<sy-tree-select clearable="true"></sy-tree-select>
```
