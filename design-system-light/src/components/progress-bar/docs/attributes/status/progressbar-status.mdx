# Status
## Definition
To set status of the progress. Three status are provided: `default`, `error`, `complete`.
Expression of status does not relate to percent attribute.<br/> For example, it is possible to set `complete` even when percent value is less than <b>100</b>.<br/><br/>
The default value is `default`.
