# Indeterminate
## Definition
Looping animation for unlimited duration. <br/>

## Boolean Attribute Handling
In the source code, <b>indeterminate = ""</b> should be modified as follows
- Set `indeterminate` attribute and don't put a value after it.
```
<sy-progress-bar indeterminate></sy-progress-bar>
```
- Alternatively, you can explicitly set it to `true`, but to not apply `indeterminate` attribute, remove it instead of setting its value to `false`.

```
<sy-progress-bar indeterminate="true"></sy-progress-bar>
```
