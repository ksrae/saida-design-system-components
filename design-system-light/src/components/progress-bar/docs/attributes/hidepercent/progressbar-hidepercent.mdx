# Hide Percent
## Definition
Determines visibility of the percent text.<br/>
The default value is `false`.

## Boolean Attribute Handling
In the source code, <b>hidePercent = ""</b> should be modified as follows
- Set `hidePercent` attribute and don't put a value after it.
```
<sy-progress-bar hidePercent></sy-progress-bar>
```
- Alternatively, you can explicitly set it to `true`, but to not apply `hidePercent` attribute, remove it instead of setting its value to `false`.

```
<sy-progress-bar hidePercent="true"></sy-progress-bar>
```
