# Closed
## Definition
Emit the <b>closed</b> event when the modal is closed.<br/>
Provide the event and value as e.detail.
- <b>event</b>: has a value of one of `ok`, `cancel`, or `close`.
- <b>value</b>: has a value if the value is included in the trigger function.
