# Trigger
## Definition
It is possible to trigger events of the Modal by calling specific function.<br/>
The Modal provides the following trigger functions:
- `setOpen()`: Opens the modal.
- `setClose(value: any)`: Closes the modal. If a value is included, it will be included in e.detail.value of the closed event.
- `setCancel(value: any)`: Triggers the event of clicking the modal's cancel button. This can be used for customizing footer to fire cancel event. If you include a value, that value will be included in the closed event's e.detail.value.
- `setOk(value: any)`: Triggers an event when the modal's ok button is clicked. This can be used for customizing footer to fire ok event. If you include a value, that value will be included in the e.detail.value of the closed event.
- `setMaximum()`: Triggers the event of maximizing the modal. This can be used for customizing footer to fire maximize event.
