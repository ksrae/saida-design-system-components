# Min
## Definition
Defines minimum number of the input number.<br/>
The value of the input number cannot be less than the minimum number. If that happens, the number will be fixed to minimum by force on blur event.<br/>
If minimum value is bigger than maximum value, maximum value is reset to default (Number.MAX_SAFE_INTEGER).<br/><br/>
Default value is `Number.MIN_SAFE_INTEGER`.
