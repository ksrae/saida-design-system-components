# Rounding
## Definition
Value of the input number is set to the nearest valid value by three options:
- round: the value of the input number rounded to the nearest integer.
- ceil: the smallest integer greater than or equal to the value of the input number.
- floor: the largest integer less than or equal to the value of the input number.


It is required to set <b>decimalPlaces attribute</b> to apply rounding.
