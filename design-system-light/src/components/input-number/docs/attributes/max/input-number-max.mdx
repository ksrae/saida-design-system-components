# Max
## Definition
Defines maximum number of the input number.<br/>
The value of the input number cannot be greater than the maximum number. If that happens, the number will be fixed to maximum by force on blur event.<br/>
If maximum value is lower than minimum value, maximum value is reset to default (Number.MAX_SAFE_INTEGER).<br/><br/>
Default value is `Number.MAX_SAFE_INTEGER`.
