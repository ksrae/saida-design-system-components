# Status
## Definition
The `status` attribute is used to indicate the validation state of the input number. It can take one of the following values:
- `success`: Indicates that the input value is success.
- `error`: Indicates that the input value is error.
- `warning`: Indicates that the input value may have issues but is not necessarily error.
- `default`: Indicates that the input value has not been validated yet or is in its default state.
