# Readonly
## Definition
The `readonly` attribute is used to make the tag component non-interactive, meaning users cannot change its value through the user interface. <br/>


## Boolean Attribute Handling
In the source code, <b>readonly = ""</b> should be modified as follows
- Set `readonly` attribute and don't put a value after it.
```
<sy-tag readonly></sy-tag>
```
- Alternatively, you can explicitly set it to true, but to not apply the `readonly`, remove the attribute instead of setting its value to false.

```
<sy-tag readonly="true"></sy-tag>
```
