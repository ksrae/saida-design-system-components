# Rounded
## Definition
Tags in rounded shape.


## Boolean Attribute Handling
In the source code, <b>rounded = ""</b> should be modified as follows
- Set `rounded` attribute and don't put a value after it.
```
<sy-tag rounded></sy-tag>
```
- Alternatively, you can explicitly set it to true, but to not apply `rounded`, remove the attribute instead of setting its value to false.

```
<sy-tag rounded="true"></sy-tag>
```
