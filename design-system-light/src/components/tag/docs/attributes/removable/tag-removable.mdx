# Removable
## Definition
To set the activation status of the tag removal option.


## Boolean Attribute Handling
In the source code, <b>removable = ""</b> should be modified as follows
- Set `removable` attribute and don't put a value after it.
```
<sy-tag removable></sy-tag>
```
- Alternatively, you can explicitly set it to true, but to not apply `removable`, remove the attribute instead of setting its value to false.

```
<sy-tag removable="true"></sy-tag>
```
