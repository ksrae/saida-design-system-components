# Overflow Count
## Definition
Determine the overflowCount value of the badge.<br/>
If the overflowCount value is greater than the value, the value will be displayed without +.<br/>
However, if the overflowCount is **less than** the value, <b>overflowCount (+)</b> will be displayed in front of the value.<br/>
Default value is <b>Infinity</b>.
