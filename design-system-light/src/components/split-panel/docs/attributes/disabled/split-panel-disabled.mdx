# Disabled
## Definition
Disables the splits. <br/>
Note that the position may still change as a result of resizing the host element.
To check this, resize the modeless popup on this page.

## Boolean Attribute Handling
In the source code, disabled = "" should be modified as follows
- Set `disabled` attribute and don't put a value after it.
```
<sy-split-panel disabled></sy-split-panel>

```
- Alternatively, you can explicitly set it to true, but to not apply `disabled`, remove the attribute instead of setting its value to false.
```
<sy-split-panel disabled="true"></sy-split-panel>

```
