# HorizontalChanged & VerticalChanged
## Definition
Events are emitted differently based on the type.
- When the type is horizontal, the <b>horizontalChanged</b> event is emitted. event.detail includes <b>leftRatio</b> and <b>rightRatio</b>.
- When the type is vertical, the <b>verticalChanged</b> event is emitted. event.detail includes <b>topRatio</b> and <b>bottomRatio</b>.
