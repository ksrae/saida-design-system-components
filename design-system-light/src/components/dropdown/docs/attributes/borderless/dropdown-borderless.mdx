# Borderless
## Definition
Removes border of the title of the dropdown. <br/>
Borderless is useful when space is limited or when a more seamless integration with the content is desired. 

## Boolean Attribute Handling
In the source code, borderless = "" should be modified as follows
- Set `borderless` attribute and don't put a value after it.
```
<sy-dropdown borderless></sy-dropdown>
```

- Alternatively, you can explicitly set it to true, but to not apply `borderless`, remove the attribute instead of setting its value to false.
```
<sy-dropdown borderless="true"></sy-dropdown>
```
