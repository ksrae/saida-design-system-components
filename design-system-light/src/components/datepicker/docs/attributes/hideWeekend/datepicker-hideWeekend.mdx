# HideWeekend
## Definition
The `hideWeekend` attribute is used to hide weekends (Saturday and Sunday) in the datepicker, making it easier to select weekdays only. This is useful for applications where weekends are not relevant, such as business scheduling or work-related tasks.

## Boolean Attribute Handling
In the source code, hideWeekend = "" should be modified as follows. <br/>
Set the <b>hideWeekend</b> attribute without assigning a value to it. <br/>
```
<sy-datepicker hideWeekend></sy-datepicker>
```
Alternatively, you can explicitly set it to true, but to not apply hideWeekend, just remove the attribute instead of setting its value to false.

```
<sy-datepicker hideWeekend="true"></sy-datepicker>
```
