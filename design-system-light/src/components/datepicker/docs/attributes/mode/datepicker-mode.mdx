# Mode
## Definition
The Mode Attribute sets the shape of the DatePicker's calendar.
If mode is 'year', the calendar will display the year calendar.

## values
- `year`: Displays the year calendar.
- `month`: Displays the month calendar.
- `day`: Displays the day calendar.
