# Form Integration / FormData

`sy-datepicker` is a **form-associated custom element**. It works inside any plain HTML
`<form>` regardless of the surrounding frontend framework:

- Submits its value under the given `name` — picked up by `FormData`.
- Honours `required` via native constraint-validation.
- Responds to `<form>.reset()` (via `formResetCallback`) and disabled fieldsets.
- Supports state restoration on back/forward navigation.

For range variants, the value is submitted as a JSON string `{"start":"…","end":"…"}`.
