# Changed Event
## Definition
The datepicker emits <b>changed</b> event when user input the date.<br/>
The changed event normally returns what the user has entered.<br/>
If variant is set to <b>range</b>, the event will include a `mode` property indicating whether the change is for the start or end date.

```
event: { 
  detail: { 
    value: string,
    mode?: 'day' | 'month' | 'year'
  }
}
```
