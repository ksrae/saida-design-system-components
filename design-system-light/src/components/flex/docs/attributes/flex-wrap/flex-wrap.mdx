# Flex Wrap 
## Definition
The `wrap` attribute defines the direction of the flex container's children.

## Values
- wrap: When the type is wrap, if the flex item does not have a width assigned, the default is flex: 1.
- nowrap: When the type is nowrap, if the flex item does not have a width assigned, the default is flex: 1.
