# Flex Justify
## Definition
The `justify` attribute defines the alignment of the flex container's children along the main axis.

## Values
- start: Items are aligned at the start of the main axis.
- center: Items are aligned at the center of the main axis.
- end: Items are aligned at the end of the main axis.
- space-between: Items are evenly distributed in the main axis.
