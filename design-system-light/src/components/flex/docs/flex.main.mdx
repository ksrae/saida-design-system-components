# Flex
## Definition
The `Flex` element is typically used to manage the layout of its children with a flexible and responsive structure.<br/>
It is based on the CSS Flexbox layout model, which helps in distributing space within a container and aligning items in a predictable way. <br/><br/>
The box default alignment of the DC component Flex is Start (Top & Left).<br/>
If you want to match align based on the text, please select Baseline in the align properties.<br/><br/>
Columns divide the page into equal vertical sections, and are used to organise content. <br/>
You can use Direction Properties to change to vertical.
RowGap are the RowGaps between the columns, and separate content in a consistent way.<br/>
columnGap are the columnGaps between the columns, and separate content in a consistent way.<br/>
Paddings define the outer edges of the grid, and prevent content from spilling out over the viewable regions.

## Fluid
The fluid grid stretches across the screen to fill all available space in the main content area. <br/>
This means the columns and the content placed within them scale and resize to fill the available space.<br/><br/>
If you want to use the fluid, apply "flex:1" to the div class="flex-item" property in sy-flex.<br/>

```
<style>
    .flex-item{
        flex:1;
    }
</style>
```
