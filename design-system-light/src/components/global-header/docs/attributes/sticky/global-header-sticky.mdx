# Sticky
## Definition
Enables a sticky header that remains visible when scrolling. Improve navigation for long pages. 

## Boolean Attribute Handling
In the source code, sticky = "" should be modified as follows
- Enter `sticky` and do not put a value after it.
```
<sy-button sticky></sy-button>
```
- Alternatively, you can explicitly set it to true, but to not apply the `sticky` attribute, simply remove it instead of setting its value to false.

```
<sy-button sticky="true"></sy-button>
```
