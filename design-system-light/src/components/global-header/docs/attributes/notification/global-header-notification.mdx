# Notification
## Definition
Enables a notification button that can be used to display alerts or messages to the user.

## Boolean Attribute Handling
In the source code, notification = "" should be modified as follows
- Enter `notification` and do not put a value after it.
```
<sy-button notification></sy-button>
```
- Alternatively, you can explicitly set it to true, but to not apply the `notification` attribute, simply remove it instead of setting its value to false.

```
<sy-button notification="true"></sy-button>
```
