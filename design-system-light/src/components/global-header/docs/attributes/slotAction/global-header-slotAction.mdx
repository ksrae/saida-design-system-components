# Slot Actions
## Definition
Use <b>slot="actions"</b> to add action area to the header.
Can include icons or text-based actions.<br/>
<i>It is recommended that the actions buttons be placed in the following order from left to right.</i><br/>
- Search
- Other
- Help 
- Notifications  
- Account
