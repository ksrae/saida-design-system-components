# Information
## Definition
The information button is visible.

## Boolean Attribute Handling
In the source code, information = "" should be modified as follows
- Enter `information` and do not put a value after it.
```
<sy-button information></sy-button>
```
- Alternatively, you can explicitly set it to true, but to not apply the `information` attribute, simply remove it instead of setting its value to false.

```
<sy-button information="true"></sy-button>
```
