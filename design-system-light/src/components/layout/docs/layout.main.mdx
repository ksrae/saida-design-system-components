# Layout 
## Definition
The Layout provides the fundamental structure for applications within our ecosystem.<br/> 
It helps maintain visual consistency across services while allowing flexible customization for each service’s unique needs.<br/>
You can add a flex inside the layout.

sy-theme is default class.<br/>
If you use theme, please add sy-theme-light or sy-theme-dark in body tag of your index.html<br/>
please check the below tag.<br/><br/>
Light Theme : body class="<b>sy-theme sy-theme-light</b>"<br/>
Dark Theme : body class="<b>sy-theme sy-theme-dark</b>"

## Attribute 

### Size 
To set the breakpoints of the responsive layout, use size attribute.<br/>
It can be applicable in <b>header</b>(`sy-header`), <b>sider</b>(`sy-sider`), and <b>footer</b>(`sy-footer`) tag.<br/>
In addition to <b>s, m, l</b>, you can specify a precise size as a string in <b> pixels</b> (e.g. size="200px").
Note : 0px is not allowed. It will be transformed to 1px.
