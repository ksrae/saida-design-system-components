# Position event
## Definition
Emit the <b>position</b> event when the modeless is moved or resized.<br/>


```
event: {
  detail: {
    top: number; // The top position of the modeless
    left: number; // The left position of the modeless
    width: number; // The width of the modeless
    height: number; // The height of the modeless
  }
}
```
