# Modeless
## Definition
Modeless is a UI that provides independent dialogs. <br/>
While Modal is a UI specialized in providing information,
modeless is more specialized in allowing users to perform additional tasks or monitoring. <br/>
<b>sy-modeless-group</b> provides various functions to easily create and control multiple modeless dialogs through script.

# Modeless group
## Definition
A modeless group allows users to manage and interact with several modeless dialogs simultaneously. <br/>
This is particularly useful in complex applications where multiple dialogs need to be open at the same time without blocking the main application workflow. <br/>
Each dialog in the modeless group can be moved, resized, and interacted with independently. <br/>
<b>sy-modeless-group</b> provides various functions to easily create and control multiple modeless dialogs through script.<br/>
For more details on the functions provided by the modeless group, please refer to the "Events" page.
