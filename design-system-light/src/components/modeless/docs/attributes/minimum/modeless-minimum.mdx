# Minimum
## Definition
Minimizes the Modeless. <br/>
The minimum Modeless leaves only the header and is displayed at the bottom of the current screen.<br/><br/>

The `minimizable` attribute must be set.

## Boolean Attribute Handling
In the source code, minimum = "" should be modified as follows. <br/>
Set the <b>minimum</b> attribute without assigning a value to it. <br/>
```
<sy-modeless minimum></sy-modeless>
```
Alternatively, you can explicitly set it to true, but to not apply `minimum`, just remove the attribute instead of setting its value to false.

```
<sy-modeless minimum="true"></sy-modeless>
```
