# Resizable
## Definition
Activates resizing the Modeless by dragging its corners or edges.

## Boolean Attribute Handling
In the source code, resizable = "" should be modified as follows. <br/>
Set the <b>resizable</b> attribute without assigning a value to it. <br/>
```
<sy-modeless resizable></sy-modeless>
```
Alternatively, you can explicitly set it to true, but to not apply `resizable`, just remove the attribute instead of setting its value to false.

```
<sy-modeless resizable="true"></sy-modeless>
```
