# Draggable
## Definition
Determines whether to allow the position of the Modeless to be changed by dragging its header.

## Boolean Attribute Handling
In the source code, draggable = "" should be modified as follows. <br/>
Set the <b>draggable</b> attribute without assigning a value to it. <br/>
```
<sy-modeless draggable></sy-modeless>
```
Alternatively, you can explicitly set it to true, but to not apply `draggable`, just remove the attribute instead of setting its value to false.

```
<sy-modeless draggable="true"></sy-modeless>
```
