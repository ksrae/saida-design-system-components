# Closable
## Definition
Determines the visibility of the <b>close</b> button.<br/>
Even if there is no close button on the modeless, it can be closed by using the `setClose()` function or by removing the `open` attribute.

## Boolean Attribute Handling
In the source code, closable = "" should be modified as follows. <br/>
Set the <b>closable</b> attribute without assigning a value to it. <br/>
```
<sy-modeless closable></sy-modeless>
```
Alternatively, you can explicitly set it to true, but to not apply `closable`, just remove the attribute instead of setting its value to false.

```
<sy-modeless closable="true"></sy-modeless>
```
