# Minimizable
## Definition
Determines the visibility of the <b>Minimum button</b>.<br/>
Even if there is no minimum button on the modeless, it can be minimized by using the `setMinimum()` function or by adding the `minimum` attribute.

## Boolean Attribute Handling
In the source code, minimizable = "" should be modified as follows. <br/>
Set the <b>minimizable</b> attribute without assigning a value to it. <br/>
```
<sy-modeless minimizable></sy-modeless>
```
Alternatively, you can explicitly set it to true, but to not apply `minimizable`, just remove the attribute instead of setting its value to false.

```
<sy-modeless minimizable="true"></sy-modeless>
```
