# Edge
## Definition
Enables edge detection when dragging or resizing.

## Boolean Attribute Handling
In the source code, edge = "" should be modified as follows. <br/>
Set the <b>edge</b> attribute without assigning a value to it. <br/>
```
<sy-modeless edge></sy-modeless>
```
Alternatively, you can explicitly set it to true, but to not apply `edge`, just remove the attribute instead of setting its value to false.

```
<sy-modeless edge="true"></sy-modeless>
```
