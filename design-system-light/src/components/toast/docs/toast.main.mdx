# Toast Message
## Definition
A toast message appears as a small, unobtrusive popup, typically in the corner of the screen, for a brief period before disappearing automatically.<br/>
It is used for temporary, system-wide notifications (e.g., action confirmations, non-blocking alerts).

Use a toast message for temporary, non-blocking feedback.<br/>
Avoid for critical or persistent alerts.
