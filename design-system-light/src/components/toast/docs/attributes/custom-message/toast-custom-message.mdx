# Custom Message

## Definition
Sets the message to be displayed in the toast message.

## Custom by slot
The toast message is opened with a predefined message in the slot. 
As long as the slots exists, there is no need to define a separate message parameter when opening.


## Custom by Function
The message through a parameter must be defined whenever opening the Toast message as a function.
