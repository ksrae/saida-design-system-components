# Variant
## Definition
To set style of the toast message.
Here's an example of each type of toast message.

- <b>info (Default)</b> : Provides additional details or general information that is relevant but not urgent
- <b>success</b> : Confirms that an action or process was completed successfully. 
- <b>warning</b> : Alerts users about potential issues that could impact their actions or outcomes, but does not indicate an error.
- <b>error</b> : Notifies users of a failure or issue that requires immediate attention or corrective action.
- <b>neutral </b> : Communicates non-critical, passive information that doesn't require immediate attention.
Use <b>slot="icon"</b> in the desired tag to insert it as an icon. <br/>
The `iconSlot` will be applied only when the variant is <b>neutral.</b>
