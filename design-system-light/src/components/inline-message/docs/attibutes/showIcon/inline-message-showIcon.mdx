# ShowIcon 
## Definition
Option to include an icon next to the message. <br/>
This helps to visually differentiate the message types or provide additional context through icons.<br/>
<b>The icon is determined by the variant of the inline message.</b>

## Boolean Attribute Handling
In the source code, showIcon = "" should be modified as follows
- Set `showIcon` attribute and don't put a value after it.

```
<sy-inline-message showIcon></sy-inline-message>
```
- Alternatively, you can explicitly set it to true, but to not apply `showIcon` attribute, remove it instead of setting its value to false.
```
<sy-inline-message showIcon="true"></sy-inline-message>
```
