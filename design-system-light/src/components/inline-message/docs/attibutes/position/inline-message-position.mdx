# Position
## Definition
There are four positions for inline messages: `bottom`, `top`, `right`, and `left`.

### Bottom (default)
If there is insufficient space below due to scrolling down or browser resizing, the message will be displayed above.<br/>
(if there is space available at the top).

### Top 
If there is insufficient space above due to scrolling down or browser resizing, the message will be displayed below.<br/>
(if there is space available at the bottom).


### Right 
If there is insufficient space on the right due to scrolling down or browser resizing, the message will be displayed on the left.<br/>
(if there is space available on the left).


### Left 
If there is insufficient space on the left due to scrolling down or browser resizing, the message will be displayed on the right.<br/>
(if there is space available on the right).
