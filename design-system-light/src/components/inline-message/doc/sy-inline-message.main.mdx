# Inline Message
## Definition
An inline message appears as a type of contextual message that appears near the element that triggers it, providing feedback, guidance, or information directly related to a user action or interaction.
Use an inline message for contextual feedback tied to specific elements or actions.<br/>
Avoid for global or temporary notifications.
