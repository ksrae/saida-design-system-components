# Custom
## Definition
To customize the button as a button-style dropdown, include two buttons in a sy-button-group.<br/>
Then add sy-dropdown into the second button to implement a button-style dropdown.<br/><br/>
Furthermore, button-group can be used to customize buttons in a number of ways.
