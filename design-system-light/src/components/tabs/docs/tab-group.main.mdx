# TabGroup
## Definition
Tab groups consist of `sy-tab` tags and `sy-tab-content` tags. <br/>
<i>Tabs must be grouped in a division with <b>slot="tabs"</b>, and the tab content must be grouped in a division with <b>slot="contents"</b></i>. <br/>
Each tab content must use a name attribute that matches the key of the corresponding tab.<br/><br/>
You can put padding on both ends of the `sy-tab`.<br/>
However, there is no padding inside the `sy-tab-content`<br/>
Please implement freely using div and `sy-flex` inside slot.<br/><br/>
