# Draggable
## Definition
All tabs can be repositioned by dragging.<br/>
The order event is fired when the position is changed.<br/><br/>
The default value is false.

## Boolean Attribute Handling
In the source code, <b>draggable = ""</b> should be modified as follows
- Set `draggable` attribute and don't put a value after it.
```
<sy-tab-group draggable></sy-tab-group>
```

- Alternatively, you can explicitly set it to true, but to not apply `draggable`, remove the attribute instead of setting its value to false.
```
<sy-tab-group draggable="true"></sy-tab-group>
```
