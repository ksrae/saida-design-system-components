# Padding
## Definition
Sets padding to the tabs. <br/>
There are three types: `small`, `medium`, `large`, and `none`<br/><br/>
`small` : 8px padding<br/>
`medium` : 12px padding<br/>
`large` : 16px padding<br/>

The default value is `none`.<br/>
