# Manual Closed
## Definition
Set manualclose if confirmation via a modal before closing a tab, or similar behavior is required.<br/>
Tabs with the manualclose property will not be closed by clicking close icon, but the closed event will be still fired.<br/>
This tab is able to be closed via the closeTab() function, unlike tabs without the closable property.
