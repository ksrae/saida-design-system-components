# SnapToMarks

## Definition
Set <b>snapToMarks</b> to `true` to snap the slider to the nearest mark when the user releases the slider handle.<br/>
This helps to restrict the slider to the defined marks.
The custom marks are defined by the <b>marks</b> property.
