# Disabled
## Definition
To set the slider disabled.

## Boolean Attribute Handling
In the source code, <b>disabled = ""</b> should be modified as follows
- Set `disabled` attribute and don't put a value after it.
```
<sy-slider disabled></sy-slider>
```
- Alternatively, you can explicitly set it to true, but to not apply `disabled`, remove the attribute instead of setting its value to false.

```
<sy-slider disabled="true"></sy-slider>
```
