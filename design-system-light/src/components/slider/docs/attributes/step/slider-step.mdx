# Step
	
## Definition
The granularity with which the slider can step through values. <br/> 
The min prop serves as the origin for the valid values. We recommend `(max - min)` to be evenly divisible by the step.<br/>
The tooltip value follows the step unit, and sliders with small values are expressed in exponential notation.
