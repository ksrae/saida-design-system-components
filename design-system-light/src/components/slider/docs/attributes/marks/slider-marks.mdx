# Marks

## Definition
The marks are displayed at the specified step interval and can be used to indicate the values that the slider can be set to.<br/>
The type of key must be number, and must in closed interval [min, max], each mark can declare its own style.<br/>
Set `snapToMarks` to true to snap the slider to the nearest mark.
