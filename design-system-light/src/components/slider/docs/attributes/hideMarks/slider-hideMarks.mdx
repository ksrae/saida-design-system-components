# HideMarks

## Definition
Hide marks on the slider. <br/>
This is useful when you want to create a cleaner look or when the marks are not necessary for the user to understand the slider's value.

## Boolean Attribute Handling
In the source code, <b>hideMarks = ""</b> should be modified as follows
- Set `hideMarks` attribute and don't put a value after it.
```
<sy-slider hideMarks></sy-slider>
```
- Alternatively, you can explicitly set it to true, but to not apply `hideMarks`, remove the attribute instead of setting its value to false.

```
<sy-slider hideMarks="true"></sy-slider>
```
