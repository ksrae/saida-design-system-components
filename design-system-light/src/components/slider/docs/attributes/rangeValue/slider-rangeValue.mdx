# RangeValue

## Definition
Defines the <b>default</b> value of each value when the range is set.<br/>
Number array with two values, the first value is the start of the range and the second value is the end of the range.<br/>
The <b>range</b> attribute must be set to <b>true</b> for this attribute to have any effect.<br/>
If you want to get the current value of the slider after user interaction, you can use the <b>change</b> event.<br/>
