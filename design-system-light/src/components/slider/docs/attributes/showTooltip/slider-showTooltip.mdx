# ShowTooltip

## Definition
Whether the tooltip should be displayed when the user hovers over the slider.<br/>
The <b>showTooltip</b> property can be set to `default`, `always`, or `never`. 
The default value is `default`.

- <b>default</b>: The tooltip is displayed when the user hovers over or drags the slider.
- <b>always</b>: The tooltip is always displayed.
- <b>never</b>: The tooltip is never displayed.
