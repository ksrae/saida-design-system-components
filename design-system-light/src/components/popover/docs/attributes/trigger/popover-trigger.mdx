# Trigger
## Definition
Sets event to open the popover.<br/> 
There are options `hover`, `click`, `focus` and `null`.<br/>
If set to `null`, popover does not have an automatic trigger and must be controlled programmatically.<br/><br/>
The default value is `hover`.
