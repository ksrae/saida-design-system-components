# Open
## Definition
Determines whether the popover is visible without any events.<br/>
If set to `true`, the popover will be displayed by default.

## Boolean Attribute Handling
In the source code, <b>open = "" </b>should be modified as follows
- Set `open` attribute and don't put a value after it.
```
<sy-popover open></sy-popover>
```
- Alternatively, you can explicitly set it to true, but to not apply `open`, remove the attribute instead of setting its value to false.

```
<sy-popover open="true"></sy-popover>
```
