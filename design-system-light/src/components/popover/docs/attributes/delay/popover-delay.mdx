# Delay
## Definition
Delay in milliseconds.<br/>
Use <b>opendelay</b> attribute to set delay before popover is shown on the event.<br/>
Use <b>closedelay</b> attribute to set delay before hidden on the event. <br/><br/>
Default value of opendelay is <b>150ms</b>.<br/>
Default value of closedelay is <b>100ms</b>.<br/>
