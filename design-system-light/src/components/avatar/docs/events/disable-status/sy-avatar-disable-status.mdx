# disableStatus

The `disableStatus` event fires whenever the `disabled` prop changes.
The event detail contains `{ disabled: boolean }`.

```js
const avatar = document.querySelector('sy-avatar');
avatar.addEventListener('disableStatus', (e) => {
  console.log(e.detail.disabled);
});
```
