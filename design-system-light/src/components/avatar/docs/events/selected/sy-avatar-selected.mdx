# selected

The `selected` event fires when a clickable avatar is clicked.
The event detail contains `{ letter, text, icon, image }`.

```js
const avatar = document.querySelector('sy-avatar');
avatar.addEventListener('selected', (e) => {
  console.log(e.detail);
});
```
