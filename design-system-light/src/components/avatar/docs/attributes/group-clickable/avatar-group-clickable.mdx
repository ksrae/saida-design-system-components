# Avatar Group Clickable
## Definition
Determines whether the avatar-group is clickable or not.<br/>
A <b>selected</b> events will be fired. <br/>
Default value is false.


## Boolean Attribute Handling
In the source code, clickable = "" should be modified as follows
- Set "clickable" attribute and don't put a value after it.
```
<sy-avatar-group clickable></sy-avatar-group>
```
- Alternatively, you can explicitly put true but to not to applied "clickable" remove the attribute not setting value to false.

```
<sy-avatar-group clickable="true"></sy-avatar-group>
```
