# Variant
## Definition
Variant attribute is to set background color of the avatar.<br/>
The color variant is : `lightgray` , `red` , `orange` , `yellow` , `lime` , `green` , `orange` , `teal` , `blue` , `purple` , `magenta` , `darkgray`.<br/><br/>
The default value is `gray`.<br/>
