# Avatar Text
## Definition
It is typically used for the initials of the username.
If the text exceeds two characters, only the first two characters will be displayed. <br/>
If the text has space like first and last names, only the first letter of each will be shown.<br/>
If an image, icon, or letter is present, the text will not be applied.
