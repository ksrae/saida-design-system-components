# Position
## Definition
Sets tooltip trigger mode.<br/> 
There are options `hover`, `click`, `focus` and `none`.<br/>

`Hover` (default)<br/> 
Show the tooltip when the user hovers over the target element.

`Click`<br/> 
Show the tooltip when the target element is clicked.


`Focus`	<br/> 
Show the tooltip when the target element receives focus (keyboard or screen reader).


`none`<br/> 
Show and hide the tooltip programmatically. Set opened property to true or false to show or hide the tooltip, respectively.
