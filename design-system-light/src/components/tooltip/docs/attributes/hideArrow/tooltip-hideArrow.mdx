# HideArrow
## Definition
Hide default arrow of your tooltip with the <b>hideArrow = ""</b> attribute. <br/>

## Boolean Attribute Handling
In the source code, <b>hideArrow = ""</b> should be modified as follows

- Set <b>hideArrow</b> attribute and don't put a value after it.
```
<sy-tooltip hideArrow></sy-tooltip>
```

- Alternatively, you can explicitly set it to true, but to not apply <b>hideArrow</b>, remove the attribute instead of setting its value to false.
```
<sy-tooltip hideArrow="true"></sy-tooltip>
```
