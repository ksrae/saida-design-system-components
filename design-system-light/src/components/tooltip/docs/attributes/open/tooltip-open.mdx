# Open
## Definition
Determines whether the tooltip itself is shown or hidden.<br/>
If open is set to true, the tooltip will not appear at all, even if the user interacts with the trigger element. <br/>

## Boolean Attribute Handling
In the source code, <b>open = ""</b>  should be modified as follows
- Set <b>open</b> attribute and don't put a value after it.
```
<sy-tooltip open></sy-tooltip>

```
- Alternatively, you can explicitly set it to true, but to not apply <b>open</b> , remove the attribute instead of setting its value to false.

```
<sy-tooltip open="true"></sy-tooltip>
```
