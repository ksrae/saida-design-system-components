# Content
## Definition
The text or message displayed inside the tooltip. <br/>
Keep it brief, as tooltips should not display lengthy content.

## Special character handling
content does not support HTML tags. Html tags will be displayed as plain text. <br/>
However, It supports special characters such as newline and tab.
- \n : newline
- \t : tab

example:
- \n: "Hello\nWorld" -> There's a newline after "Hello," followed by "World."
- \t: "Hello\tWorld" -> There's a tab after "Hello," followed by "World."
