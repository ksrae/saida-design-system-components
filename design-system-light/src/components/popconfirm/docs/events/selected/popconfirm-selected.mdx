# Selected
## Definition
Clicking any button emits <b>the selected</b> event.<br/><br/>
If popconfirm has closable attribute, clicking outside of the popconfirm will emit the selected event. <br/>
The emitted value is set to `cancel` type.
and <b> event.detail</b> includes the type of the clicked button.
