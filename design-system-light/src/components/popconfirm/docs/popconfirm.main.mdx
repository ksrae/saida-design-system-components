# Popconfirm
## Definition
Popconfirm is similar to a popover. Popconfirm is a popover modal which means it opens near the parent DOM element but has confirm buttons in it.<br/>
By default, popconfirm doesn't close without user response.<br/>

This is useful for getting a user's response to a simple request. <br/>
