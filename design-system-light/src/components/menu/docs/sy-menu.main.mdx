# Menu
## Definition
Menu displays menu-items by trigger: `click`, `hover`, or `contentmenu`.<br/>
Menu provides several tags: <b>menu-sub, menu-item, menu-group</b>.

- `menu-item`: displays value that a user is defined. It emits selected event when is clicked.
- `menu-sub`: opends new menu-box and displays menu-items that belongs to.
- `menu-group`: displays menu-items that belongs to menu-group as a children in the menu-box.
