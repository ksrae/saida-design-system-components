# Trigger
## Definition
Sets Menu trigger mode.<br/> 
There are options `hover`, `click`, and `contextmenu`.<br/>
`contextmenu` is to open menu with right click<br/><br/>
Default value is `hover`.<br/>
However, when the menu belongs to dropdown, this trigger is fixed to `click`.<br/>
