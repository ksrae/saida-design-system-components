# Checkable
## Definition
Determines whether add checkbox to all the menu items or not.<br/>
If set true, the menu does not close when any item is selected or checked.

## Boolean Attribute Handling
In the source code, checkable = "" should be modified as follows
- Set `checkable` attribute and don't put a value after it.
```
<sy-menu checkable></sy-menu>
```

- Alternatively, you can explicitly set it to true, but to not apply `checkable`, remove the attribute instead of setting its value to false.
```
<sy-menu checkable="true"></sy-menu>
```
