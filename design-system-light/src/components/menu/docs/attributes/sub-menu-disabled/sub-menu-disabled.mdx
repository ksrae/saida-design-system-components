# Sub Menu Disabled
## Definition
Disables sub menu. Disabled sub menu will not open its children.


## Boolean Attribute Handling
In the source code, disabled = "" should be modified as follows
- Set `disabled` attribute and don't put a value after it.
```
<sy-menu-sub disabled></sy-menu-sub>
```

- Alternatively, you can explicitly set it  to `true`, but to not apply the `disabled`, remove the attribute instead of setting its value to`false`.
```
<sy-menu-sub disabled="true"></sy-menu-sub>
```
