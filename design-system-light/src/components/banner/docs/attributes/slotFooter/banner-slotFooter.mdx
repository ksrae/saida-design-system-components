# Slot Footer
## Definition
Custom footer slot in the banner message. If not set, the footer will not be displayed. <br/>
To set the footer using the slot, add the <b>slot="footer"</b> attribute to the tag. <br/>

The following example inserts two <b>sy-button</b> elements into the <b>slotFooter</b> and handles the click events from each button, `btn1` and `btn2`.
